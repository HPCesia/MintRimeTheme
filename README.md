# Rime Theme

Rime Theme 是一个为 Rime 输入法引擎提供第三方主题的仓库，是 [oh-my-rime|薄荷输入法](https://github.com/ibupliz/oh-my-rime) 项目的补充。

oh-my-rime 自带了鼠须管和小狼毫的主题皮肤，但是Android的同文输入法和仓输入法比较特殊，所以另外建立此仓库。

## 主题预览

该仓库收录了的主题，尽可能匹配oh-my-rime内的黑水鸭和蓝水鸭样式：
